# NW37 - Installing

Windows:
- run '_installer.bat'
- run 'run.bat'

Linux:
- sudo git clone https://www.github.com/KingfernJohn/NW37.git
- sudo pip install termcolor
- sudo pip install decimal
- cd NW37
- cd src
- sudo python3 cc.py
